# ExcitedTau
the search for excited tau leptons...

## To install and run on a test file:
* `cmsrel CMSSW_10_2_5`
* `cd CMSSW_10_2_5/src/`
* `cmsenv`
* `git clone https://github.com/fojensen/ExcitedTau`
* `cd ExcitedTau/`
* `scram build`
* `cmsRun excitingAnalyzer_cfg.py isSignalMC=True >& log &`
